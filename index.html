my-playlist

minimal youtube playlist controller. configured using url parameters. if you have many playlists, you can configure a url and save it as a bookmark for easy playing.

parameters

add the parameters below to the url and refresh the page, like this: https://kiwijuice56.github.io/my-playlist/?id=PL8F6B0753B2CCA128&shuffle=1. afterwards, you can play, pause, and skip using keyboard controls.

  • id=[PLAYLIST_ID]: plays the given playlist. locate PLAYLIST_ID by opening the playlist on youtube and copying the text after ?list=
  • shuffle=1: shuffles the given playlist
